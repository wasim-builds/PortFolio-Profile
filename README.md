# Mohammed Wasim Khan — Developer Portfolio

A premium developer portfolio showcasing open-source contributions, projects, and technical expertise across AI/ML, full-stack development, and cloud infrastructure.

## Features

- **Open Source Contributions:** Auto-populated cards for 16+ major projects (Apple, AWS, Google, Microsoft, PyTorch, etc.) with individual PR listings
- **Project Filters:** Filterable project grid (All / Featured / AI / Web / Other)
- **Technical Skills:** 5 categorized skill groups with proficiency bars and icons
- **Experience Timeline:** Visual timeline with color-coded dots for work, research, and education
- **Premium Dark UI:** Deep dark luxury theme with warm orange accent and micro-interactions
- **Custom Cursor:** Smooth trailing cursor with hover expansion on interactive elements
- **Scroll Reveal:** Intersection Observer-powered reveal animations
- **Mobile Navigation:** Hamburger menu for mobile/tablet with full-screen overlay
- **Accessibility:** `prefers-reduced-motion` support, ARIA labels, semantic HTML
- **Page Loader:** Elegant loading animation with progress bar

## Tech Stack

- **Fonts:** [Satoshi](https://api.fontshare.com/v2/css?f[]=satoshi@900,700,500,400) (headings) + [Inter](https://fonts.google.com/specimen/Inter) (body)
- **HTML5** — Semantic markup with ARIA attributes
- **CSS3** — Custom properties, CSS Grid, Flexbox, animations, `prefers-reduced-motion`
- **Vanilla JavaScript** — ES6 modules, Intersection Observer API, dynamic rendering

## Deployment

This project is statically generated and can be deployed via Vercel, Netlify, or GitHub Pages.

```bash
python3 -m http.server 8080
# Then open http://localhost:8080
```
