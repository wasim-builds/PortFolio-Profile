# Mohammed Wasim Khan - Premium Developer Portfolio

A high-end, performance-optimized software engineering portfolio designed to showcase my projects, technical expertise, and system architecture skills. 

## Features
- **Premium UI/UX:** Built with a dark-themed glassmorphism aesthetic and ambient background lighting.
- **Dynamic Content:** Projects and technical skills are dynamically injected using modular JavaScript for easy maintenance and scalability.
- **Interactive Loading Screen:** Features a sleek, interactive entry screen with a mouse-tracking glow effect.
- **Responsive Design:** Fully responsive layout optimized for mobile, tablet, and desktop viewing.

## Tech Stack
- HTML5
- CSS3 (Custom Variables, Flexbox/Grid, Animations)
- Vanilla JavaScript (ES6 Modules, Intersection Observer API)
- Font: [Geist](https://fonts.google.com/specimen/Geist)

## Deployment
This project is statically generated and can be easily deployed via Vercel, Netlify, or GitHub Pages.
